# Base de Conhecimento de Tecnologias

Uma coleção curada de linguagens de programação, frameworks e ferramentas com links para suas documentações oficiais.

| Tecnologia | Descrição | Tags |
| :--- | :--- | :--- |
| [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) | JavaScript é uma linguagem de programação da Web leve, interpretada e baseada em objetos. É amplamente utilizada para desenvolvimento web, permitindo a criação de páginas interativas e dinâmicas. | - |
| [Python](https://docs.python.org/pt-br/3.14/) | Python é uma linguagem de programação da ciência de dados e IA de alto nível, interpretada e de propósito geral. Conhecida por sua sintaxe clara e legibilidade, é amplamente utilizada em desenvolvimento web, ciência de dados, automação e muito mais. | - |
| [Java](https://docs.oracle.com/en/java/) | Java é uma linguagem de programação orientada a objetos, de propósito geral e baseada em classes. É uma das linguagens mais populares, especialmente para aplicações cliente-servidor web e desenvolvimento Android. | - |
| [C#](https://learn.microsoft.com/pt-br/dotnet/csharp/) | C# (pronuncia-se 'C sharp') é uma linguagem de programação moderna, orientada a objetos e de tipo seguro. Desenvolvida pela Microsoft, é amplamente utilizada para construir aplicações Windows, web e jogos com o motor Unity. | - |
| [C++](https://isocpp.org/get-started) | C++ é uma linguagem de programação de propósito geral com recursos de programação de baixo e alto nível. É uma extensão da linguagem C e é amplamente utilizada em software de sistema, jogos e aplicações de alto desempenho. | - |
| [TypeScript](https://www.typescriptlang.org/docs/) | TypeScript é um superconjunto de JavaScript que adiciona tipagem estática opcional à linguagem. Desenvolvido pela Microsoft, visa construir aplicações JavaScript em larga escala de forma mais segura e robusta. | - |
| [PHP](https://www.php.net/manual/pt_BR/) | PHP é uma linguagem de script de propósito geral, especialmente adequada para o desenvolvimento web. É usada para criar sites dinâmicos e interativos, sendo a base de sistemas como WordPress. | - |
| [Go](https://go.dev/doc/) | Go (ou Golang) é uma linguagem de programação compilada e de código aberto projetada no Google. É conhecida por sua simplicidade, eficiência e forte suporte para programação concorrente. | - |
| [Rust](https://www.rust-lang.org/pt-BR/learn) | Rust é uma linguagem de programação focada em segurança, velocidade e concorrência. Ela permite construir software confiável e eficiente, sendo uma ótima escolha para programação de sistemas. | - |
| [React](https://reactjs.org/) | Uma biblioteca front-end declarativa, eficiente e flexível para criar interfaces de usuário baseadas em componentes. | `biblioteca`, `frontend`, `ui`, `meta` |

| Next.js | Um framework de React para produção, que facilita renderização do lado do servidor (SSR), geração de sites estáticos (SSG) e full-stack development. | `framework`, `fullstack`, `ssr`, `react` |

| Swift | Uma linguagem de programação poderosa e intuitiva desenvolvida pela Apple para criar aplicativos para iOS, iPadOS, macOS, watchOS e tvOS. | `linguagem de programação`, `apple`, `ios`, `backend` |
| Scala | Uma linguagem que combina recursos de programação orientada a objetos e funcional, sendo executada na JVM e popular em Big Data. | `linguagem de programação`, `jvm`, `funcional`, `big data` |

| Docker | Uma plataforma para desenvolver, enviar e executar aplicações dentro de contêineres, garantindo ambientes isolados e consistentes. | `devops`, `containerizacao`, `virtualizacao`, `ferramenta` |
